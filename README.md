Full-Stack Airbnb-Like Web Application

Overview

Wanderlust is a full-stack web application inspired by Airbnb that allows users to explore, list, and book accommodations. The project focuses on implementing real-world features using a clean and scalable backend-driven approach.

Features

User authentication and authorization, property listings with image uploads, booking functionality, and a review system. The application supports complete CRUD operations and responsive UI behavior.

Tech Stack

Node.js, Express.js, MongoDB, Mongoose, EJS, HTML, CSS, JavaScript. Authentication is handled using session-based login and authorization middleware.

Architecture

The project follows the MVC architecture with RESTful routing, centralized error handling, and modular middleware for authentication, validation, and data management.
